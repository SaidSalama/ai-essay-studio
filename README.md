# 🎨 AI Essay Studio

An interactive AI-powered essay writing assistant with human-in-the-loop capabilities, real-time streaming, and time travel features.

## 📋 Overview

AI Essay Studio provides a transparent and controllable AI writing experience. Users can guide the AI through planning, research, drafting, and revision stages with full visibility into the agent's thought process.

## ✨ Features

- 📝 **Real-time Streaming** - Watch essays being written token by token
- 🧠 **Human-in-the-Loop** - Approve, edit, or reject AI-generated plans
- ⏰ **Time Travel** - Navigate through revision history and restore previous drafts
- 🔍 **Research Visibility** - See AI research queries as they happen
- 💬 **Interactive Critique** - Provide feedback for essay revisions

## 🛠️ Tech Stack

- **Frontend**: React 18 + TypeScript + CSS Modules
- **Backend**: LangGraph (running on Google Colab)
- **Container**: Docker with Dev Container support

## 📦 Prerequisites

- Docker Desktop
- VS Code with Dev Containers extension
